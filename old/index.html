Welcome to Speedy Review

Your go-to tool for generating commit messages based on git diffs.

How It Works

Speedy Review analyzes your git diffs and suggests concise commit messages.

Getting Started

  1. Install the necessary dependencies.
  2. Set up your environment variables.
  3. Run the script during your commit process.
